# Island Gun King

A single-page browser arcade shooter.

[![Live Demo](https://img.shields.io/badge/demo-island--gun--king.vercel.app-222)](https://island-gun-king.vercel.app)

## Overview

Stand on a tropical island, shoot waves of incoming threats, and rack up the highest score you can before the screen reads **GAME OVER**. Pure JavaScript, zero dependencies, playable in any modern browser.

## Design & Tech

- **Stack:** Single HTML file, vanilla JavaScript, Canvas
- **Aesthetic:** Retro arcade
- **Deployment:** Vercel
